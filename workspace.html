What is React?

React Native is an open-source mobile application framework created by Facebook. It is used to develop applications for Android, iOS, Web and UWP by enabling developers to use React along with native platform capabilities.
React also allows us to create reusable UI components.



Why is React JS popular?

React's popularity is increasing more and more day by day.It's now being used on more than 90,000 websites. Why so?Let's take a look -

  • React basically gives developers the ability to work with a virtual browser that is more friendly than the real browser.

  • React allows us to create reusable UI components.

  • It makes developing apps more efficient & fun.

  • React.js is a flexible and efficient front end javascript library for building UI, and is considered the most in demand in the job markets.

  • React is very flexible.
